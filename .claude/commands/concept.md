Create a new architectural concept file from template.

Usage: `/concept <short-title>`

Example: `/concept real-time-collaboration`

This command:
1. Finds the next available CONCEPT-XXX number
2. Creates a new concept file from CONCEPT-TEMPLATE.md
3. Sets the date and initial status
4. Updates the concepts/README.md index
5. Opens the file for editing

Concepts are exploratory ideas for future features, architectural patterns, or design inspirations. They are NOT commitments - just documented possibilities.
